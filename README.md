# MoonBit in HTML

VSCode 插件，为 HTML 文件中的 MoonBit 代码提供语法支持。

## 功能

- 在 HTML 文件中支持 MoonBit 语法高亮
- 支持在 `<script type="mbt">` 标签中编写 MoonBit 代码

## 安装

您可以通过以下方式安装此插件：

1. 从 VS Code 市场安装
   - 打开 VS Code
   - 按下 `Ctrl+P`
   - 输入 `ext install oboard.moonbit-html`

2. 从 VSIX 文件安装
   - 下载最新的 `.vsix` 文件
   - 在 VS Code 中，选择 "从 VSIX 安装..."
   - 选择下载的文件进行安装
